Hello this is the Quake Tactics creator. to play the mod you will 
go into the modAssets folder grab the zip put it in your quak4 
game folder and extract it and you should be able to play. 
Some basic Info 
H - End Turn Button 
M - Help Menu Button 
Click to select unit 
Click Again to 
	Move Unit if selecting the ground 
	Attack a enemy unit if selecting an enemy
	Heal if selecting a friendly unit with medic
	Repair if selecting a friendly with technician (restores turn)

Have fun and be tactical
